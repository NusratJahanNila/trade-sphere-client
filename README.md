Website Name: Trade Sphere
-----------------
Description:

Trade Sphere is a full-stack web application designed to simplify international trade operations by providing exporters and importers with a digital platform to manage, list, and track global trade products efficiently.

It offers a secure, user-friendly interface where users can explore the latest export-ready products, manage their own exports, and monitor imported goods â€” all in one place.
-------
Features:


1. Dynamic Homepage:
Includes an interactive slider, trade information section, the latest six products, and a Top-Rated Products section showcasing three high-rated exports.

2. All Products Page:
Displays all available export-ready products fetched dynamically from the database, following the layout and design described in the project documentation.

3. My Export Section (Private Route):
Logged-in exporters can view, update, or delete the products they have added for export.

4. My Import Section (Private Route):
Users can view products they have imported, along with quantity and trade details.They can also remove any product from the import page,as well as database.

5. Add Export Page (Private Route):
Allows authenticated exporters to list new export items, including product details, image URL, price, origin, and rating.

6. Firebase Authentication:
Secure login, registration, and Google authentication system integrated with Firebase for user identity management.

7. Real-Time Database Integration:
MongoDB Atlas and Express.js handle CRUD operations for products, imports, and exports with smooth data flow.
---------

Tools:

Frontend: React, React Router, Tailwind CSS, Lucide Icons, DaisyUI,Firebase Auth


Backend: Node.js, Express.js, MongoDB Atlas
